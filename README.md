# hydrogenoti
Hydrogenoti is a device that produces hydrogen from water and fills it into a balloon to build harmless hydrogen bombs (they are not radioactive).
I design it using Fusion360 and print it with PETG.

## **electronics**

I didn't make a wiring diagram because it would be just two wires :D. 
The electrodes are nine. So I had to choose to which pole 5 of them will be connected. I chose GND, because there will be more hydrogen than oxygen on the surfaces of the electrodes. The GND-electrodes are arranged in the corners and in the middle. VCC uses the remaining electrodes, of course.
